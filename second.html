Question-1: What are the different ways to select an element in the DOM?

Answer : To select a DOM element there are methods like getElementById, getElementByClassName, getElementsByTagName, querySelector, and querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : innerHTML contains the complete Html content in the element innerText returns only the content written inside the tag and textContent retrieves including html tags, content hidden content also

Question-3: What is event delegation in the DOM?

Answer : Event Delegation is a technique that efficiently handles events triggered on its child element instead of adding saparate listeners to each

Question-4: What is event bubbling in the DOM?

Answer : Event bubbling is a type of DOM that not only on that element but also on it's parent element
